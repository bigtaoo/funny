版本：v0.1  
状态：草稿，进行中

## 一、屏幕规格

| 参数 | 数值 |
|---|---|
| 目标分辨率 | 1080 × 1920（手机竖屏） |
| 格子尺寸 | 84 × 84 px（正方形） |
| 战场列数 | 12 列 |
| 战场行数 | 18 行 |
| 战场宽度 | 12 × 84 = 1008 px，左右各 36 px 边距 |
| 战场高度 | 18 × 84 = 1512 px |

## 二、整体布局（竖屏）

```
┌─────────────────────────────┐  ← 1080px
│  顶部 HUD                    │  ~100px  计时器 + 阶段指示
├─────────────────────────────┤
│                             │
│         战  场               │  ~1512px  12×18 格子
│   (含基地、建筑行、战斗区)    │
│                             │
├─────────────────────────────┤
│  资源 HUD                    │  ~100px  金币 + 升级基地按钮
├─────────────────────────────┤
│  手牌区                      │  ~208px  6 张手牌
└─────────────────────────────┘
```

总计：100 + 1512 + 100 + 208 = 1920 px

### 宽屏 PC 适配

游戏区锁定竖屏比例居中显示，左右两侧留白用笔记本装饰填充（装订线、涂鸦元素）。不扩展功能区域。

## 三、手牌区

- 6 张卡牌横向排列，均匀分布
- 卡牌尺寸：约 155 × 190 px（宽 = (1080 - 间距) / 6）
- 卡面内容：兵种/建筑/法术简笔图标 + 费用数字（手写字体）
- 状态：
  - 正常：正常显示
  - 选中：卡牌上移约 20 px，高亮描边
  - 禁用（金币不足）：涂改液白半透明遮罩，费用数字被划掉
  - 即将刷新：半透明橡皮擦纹理从底部向上覆盖，卡面内容始终可见（透过遮罩可读）

### 手牌刷新机制

- 打出一张牌 → 该槽立即随机抽新牌，新牌计时器从 2 分钟重新开始
- 2 分钟未使用 → 该槽自动随机刷新为新牌
- 牌池无限循环，不会用完
- 开局时 6 张牌的计时器随机错开（各自 0～60 秒随机偏移），防止多张同时刷新

### 刷新倒计时视觉

橡皮擦效果：米白色半透明纹理从卡牌底部向上缓慢覆盖，2 分钟走完全程。
- 半透明（约 60% 不透明度），卡面图标和费用始终可读
- 覆盖完成时短暂停顿，卡牌"重新画出"新牌（出生弹出动效）
- 刷新时不打断玩家操作，静默替换

## 四、出牌交互

所有出牌均采用**选牌 → 拖拽**手势，三类卡牌反馈机制不同，手势一致。

### 兵种卡

1. 点击手牌 → 10 条进攻路线全部高亮（绿色=可放，灰/红=出兵行已占）
2. 拖拽至目标列任意位置（只识别列，不识别行）→ 该列格子放大提示
3. 出兵行被占 → 该列变红，禁止放置
4. 松手 → 单位在出兵行（row 16）生成，开始推进，扣除金币
5. 拖出战场 → 取消

> 单位永远从出兵行出发，玩家无法指定行。

### 建筑卡

1. 点击手牌 → 己方建筑行中未放置建筑的格子显示暗色预览图标
2. 拖拽至目标格 → 预览图标变亮
3. 已有建筑的格子无提示，不可放
4. 松手 → 建筑生成，扣除金币
5. 拖出战场 → 取消

### 法术卡

1. 点击手牌 → 卡片高亮
2. 拖入战场 → 影响范围高亮（吸附最近格子中心）
   - 陨石打击：2×2 四格高亮
   - 急速冲锋：无需选格，松手立即对己方全体单位生效
3. 松手 → 法术立即生效，扣除金币
4. 拖出战场 → 取消

### 通用规则

- 金币不足：卡牌为禁用状态，不可拖拽
- 拖回手牌区或拖出战场边界：均视为取消，卡牌归位

## 五、HUD 信息层级

### 顶部 HUD（~70px）

```
[ 计时器 ]          [ 敌方基地 HP 条 ]          [ 设置按钮 ]
  左侧居中             显示在敌方基地正下方           退出/设置
```

- **左**：游戏计时，手写字体，显示 mm:ss
- **中**：敌方基地 HP 条，紧贴敌方基地（2×2 格 = 168px）下方，宽度与基地齐
- **右**：设置按钮（退出游戏等）
- 不显示当前阶段倍率（阶段变化通过基地 buff 特效表现）
- 不显示敌方金币

### 底部资源 HUD（~100px）

```
[ 金币图标 + 数字 ]   [ 己方基地 HP 条 ]   [ 升级基地按钮 + 费用 ]
     左侧                在基地正上方              右侧
```

- **左**：金币图标（简笔硬币）+ 数字，手写字体。接近上限时旁边出现「!」涂鸦提示
- **中**：己方基地 HP 条，紧贴己方基地（row 17）上方，宽度与基地齐（168px）
- **右**：升级基地按钮，显示当前升级费用（50 / 100 / 200）。金币不足时灰显
  - 防误触：长按 0.5 秒激活，松手取消，无需弹窗确认

### HP 条设计（笔记本风格）

- 形式：方格填充血条，共 10 格，每格 = 10% HP
- 满血：格子全部填充实心
- 掉血：对应格子被划叉（手绘 × 符号），伴随轻微抖动动效
- 危险（≤30%）：剩余格子描边变红

### 阶段变化提示

不在 HUD 显示倍率数字，改为基地 buff 特效：
- 产金加速（×1.5 / ×2.0 / ×4.0）：基地旁出现速度线 + 金币飞出动效
- 攻击倍率（13min，×2）：所有单位出现红色描边特效

## 六、战场单位血条

- **单位**：受击时显示血条，3 秒后淡出；未受击时不显示，保持战场清晰
- **建筑**：血条常驻显示（建筑是战略目标，玩家需要持续判断）
- 血条风格沿用笔记本美术：细长方格填充条，掉血格子划叉

---

## 七、对战匹配流程

- 所有对战统一走匹配流程，无单独"vs AI"入口
- 无可用真人玩家时，默认匹配机器人；机器人有随机名字、等级等信息，与真人外观一致
- 卡牌详情（属性、说明）**仅在大厅**可查看，游戏内不提供，避免打断操作节奏

## 八、结算界面

### 胜负显示

- 大标题显示胜/负，手写风格字体，带简单动效（胜利纸张飞溅，失败打叉）

### 表现评级徽章

双方玩家各获得一枚或多枚徽章，无论胜负。徽章为手绘风格图标 + 称号文字。

| 称号 | 判定依据 |
|---|---|
| 最佳输出 | 对敌方基地造成伤害最高 |
| 铁壁防线 | 己方基地受到伤害最少 |
| 兵海战术 | 单局派兵数量最多 |
| 建筑大师 | 建筑存活时间总和最长 |
| 精准打击 | 法术命中单位数量最多 |
| 以少胜多 | 用较少金币消灭较多敌方单位 |

### 己方最大成就

可获得多枚徽章，但大字突出显示**得分最高的那枚**。例如同时获得"最佳输出"和"精准打击"，则突出显示"最佳输出"。结算画面用醒目手写大字呈现，例如：

> **「你消灭了 47 个敌方单位」**  
> **「你的基地只受到了 120 点伤害」**

文字样式：大号手写字体，带下划线或荧光笔标注效果，强化"笔记本批注"感。

## 九、大厅 UI

### 布局方案（底部导航）

```
┌─────────────────────────────┐
│  头像 + 名字 + 等级   [ ⚙ ] │  顶部个人信息条 + 设置
├─────────────────────────────┤
│                             │
│        游戏 Logo            │
│                             │
│      涂鸦装饰区              │  中间大空间，笔记本封面感
│   （小人 / 对话框 / 图案）   │
│                             │
│  ┌───────────────────────┐  │
│  │       开始匹配         │  │  最大最突出的按钮，居中
│  │   （预计等待 <10秒）   │  │
│  └───────────────────────┘  │
├─────────────────────────────┤
│  图鉴  战绩  主页  商店  社交 │  底部导航栏（商店/社交为预留灰显）
└─────────────────────────────┘
```

- 开始匹配按钮：全宽减边距，高度约 80px，是页面视觉重心
- 涂鸦装饰区占据 Logo 下方到按钮之间的空间，强化笔记本封面感
- 底部导航 5 个图标，商店和社交暂时灰显预留
- PC 宽屏处理方式同游戏内：竖屏区域居中，两侧笔记本装饰填充

### 匹配流程

**匹配中**：不跳转新页面，原地更新开始匹配按钮状态——按钮变为"匹配中…"，带动效（如笔画循环描边或跳动省略号），点击可取消。

**匹配成功**：进入 A vs B 界面：
- 全屏展示双方头像 / 名字 / 等级，中间一个手绘风格"VS"
- 此界面同时作为游戏资源加载画面，加载完成后自动进入游戏
- 展示时长约 2～3 秒（加载时间视情况延长）
