# 核心玩法循环设计文档

版本：v0.1  
状态：草稿，待评审

---

## 一、游戏定位一句话

**实时塔防对战**：双方同时用随机手牌派兵、建造、施法，在同一张地图上推进，率先摧毁对方主基地获胜。

---

## 二、核心循环概览

```
主基地自动产金币
       ↓
玩家从手牌中选择一张出牌（消耗金币）
       ↓
       ├─ 出兵卡  → 立即在己方区域生成单位，自动向敌方推进
       ├─ 建筑卡  → 在地图指定格放置建筑，持续产兵
       └─ 法术卡  → 立即生效的一次性爆发效果
       ↓
战场上单位自动行进、自动攻击
       ↓
单位到达敌方主基地 → 扣除基地血量
       ↓
任一方基地血量归零 → 判负，游戏结束
```

**单局目标时长：5 ～ 10 分钟。**

## 三、资源系统

### 3.1 金币

| 参数 | 初始值 | 备注 |
|---|---|---|
| 基地每秒产金币 | 2 | 可通过升级基地提升 |
| 基地升级费用 | 50 / 100 / 200 | 共 3 级，每级提升产速 +1/s |
| 升级后产速 | 3 / 4 / 5 | |
| 金币上限 | 30 | 溢出不再累积，防止囤积 |

> **设计意图**：金币上限迫使玩家持续决策，不能无限等待存大钱；基地升级提供长期投资路线，与即时出牌的短期路线形成博弈。

### 3.2 金币流向优先级（玩家决策空间）

```
选择A：升级基地  → 长期收益，但前期弱势
选择B：放置建筑  → 中期收益，需要建筑存活
选择C：直接出兵  → 立即效果，无持续收益
选择D：使用法术  → 一次性爆发，改变局面
```

## 四、手牌系统

### 4.1 基本规则

- 游戏开始时，从**牌池**中随机抽取 **6 张**手牌。
- 手牌上限 6 张，打出一张后从剩余牌池补充一张（如果牌池有剩余）。
- 牌池由固定的 **12 张牌**组成（每种卡 2 张），每局随机排序。
- 手牌内容决定本局策略方向，不同开局手牌产生不同打法。

### 4.2 MVP 牌池组成（12 张）

| 类型 | 卡名 | 数量 | 费用 | 说明 |
|---|---|---|---|---|
| 兵种 | 普通兵 | 2 | 4 | 均衡单位，一次出 2 个 |
| 兵种 | 盾兵 | 2 | 6 | 坦克，吸引仇恨 |
| 兵种 | 弓箭兵 | 2 | 5 | 远程，不参与近战 |
| 建筑 | 兵营 | 2 | 10 | 每 4 秒自动产 1 个普通兵 |
| 建筑 | 箭塔 | 2 | 12 | 攻击范围内的敌方单位 |
| 法术 | 急速冲锋 | 1 | 8 | 己方所有单位移速 ×2，持续 5 秒 |
| 法术 | 陨石打击 | 1 | 12 | 对目标区域造成大量伤害，清场用 |

> **设计意图**：12 张牌、6 张手牌保证每局手牌差异显著，同时法术只有各 1 张，出现与否具有随机性，增加策略不确定性。

### 4.3 出牌交互

1. 玩家点击手牌 → 卡片高亮，进入"放置模式"。
2. 兵种卡/法术卡：点击己方区域（地图下半部分）任意格 → 立即生效，扣除金币。
3. 建筑卡：点击允许放置的格子 → 建筑出现，扣除金币。
4. 再次点击同一手牌或按 ESC → 取消放置模式。
5. 金币不足时，手牌显示变灰，无法出牌。

## 五、单位系统

### 5.1 三种基础单位（MVP）

#### 普通兵（Swordsman）

| 属性 | 数值 |
|---|---|
| HP | 60 |
| 攻击力 | 12 |
| 攻击间隔 | 1.0s |
| 移动速度 | 80 px/s |
| 攻击范围 | 近战（50px） |
| 费用 | 4（出 2 个） |

**行为**：直线向前推进，优先攻击最近的敌方单位。无特殊机制。

---

#### 盾兵（Guardian）

| 属性 | 数值 |
|---|---|
| HP | 150 |
| 攻击力 | 8 |
| 攻击间隔 | 1.5s |
| 移动速度 | 55 px/s |
| 攻击范围 | 近战（50px） |
| 费用 | 6（出 1 个） |

**行为**：移动慢，但**仇恨吸引**——同屏范围内的敌方单位优先攻击盾兵，而非其他友方单位。配合弓箭兵形成"肉盾+输出"组合。

---

#### 弓箭兵（Archer）

| 属性 | 数值 |
|---|---|
| HP | 35 |
| 攻击力 | 18 |
| 攻击间隔 | 2.0s |
| 移动速度 | 70 px/s |
| 攻击范围 | 远程（200px） |
| 费用 | 5（出 1 个） |

**行为**：**停止移动后攻击**——进入攻击范围内有敌方单位时停下来射击，而不继续推进。血量极低，需要盾兵保护。

---

### 5.2 兵种克制关系（MVP）

```
盾兵（高HP）  →  硬扛弓箭兵的远程伤害
弓箭兵（远程）→  在推进途中打击建筑和人群
普通兵（均衡）→  用数量优势压制单体盾兵
```

> **注意**：MVP 阶段无显式克制系数（如伤害加成），克制关系由行为差异自然产生，不靠数值硬算。这降低了玩家理解成本。

## 六、建筑系统

### 6.1 建筑放置规则

- 建筑只能放置在己方区域的**空格**上。
- 每条推进路线上有 **2 个建筑格**（共 2 条路线 = 4 个建筑格）。
- 建筑一旦放置，不可移动，不可回收金币。
- 建筑有 HP，可被敌方单位摧毁。

### 6.2 MVP 建筑

#### 兵营（Barracks）

| 属性 | 数值 |
|---|---|
| HP | 200 |
| 每次出兵 | 普通兵 ×1 |
| 出兵间隔 | 4 秒 |
| 费用 | 10 |

> **设计意图**：提供持续兵力来源，解放玩家手动出兵的压力，给玩家更多精力管理法术和资源。

#### 箭塔（Arrow Tower）

| 属性 | 数值 |
|---|---|
| HP | 120 |
| 攻击力 | 15 |
| 攻击间隔 | 1.5s |
| 攻击范围 | 150px（放置格为中心） |
| 费用 | 12 |

> **设计意图**：防守型建筑，阻止敌方推进。与兵营配合使用时，能有效拖住敌方单位让兵营的兵慢慢输出。

## 七、地图设计（MVP 版本）

```
╔══════════════════════════╗
║     敌方主基地  [HP:300] ║
║─────────────────────────║
║  [建筑格A1] [建筑格A2]  ║  ← 敌方建筑区
║─────────────────────────║
║        战斗区域          ║
║   路线L      路线R      ║  ← 两条推进路线
║        战斗区域          ║
║─────────────────────────║
║  [建筑格B1] [建筑格B2]  ║  ← 己方建筑区
║─────────────────────────║
║     己方主基地  [HP:300] ║
╚══════════════════════════╝
```

- **路线 L（左路）**：主路，宽敞，双方兵力正面对抗的主战场。
- **路线 R（右路）**：侧路，稍窄，偷袭或分兵压迫用。
- 兵种出牌时默认分配到左路，可选择右路。
- 建筑格固定，放置后影响对应路线的防守深度。

## 八、胜负判定与单局节奏

### 8.1 胜负条件

- 任一方主基地 HP 降至 0 → 该方立即判负，对方获胜。
- 单局无时间上限，但资源加速机制会推动游戏结束。

### 8.2 资源加速（防止拉锯）

| 游戏时间 | 基地产金币速度 |
|---|---|
| 0 ～ 3 分钟 | 正常（基础速度） |
| 3 ～ 6 分钟 | ×1.5 |
| 6 分钟以上 | ×2.0 |

> **设计意图**：时间越长，双方资源越充裕，单位越多，战场越混乱，自然推向决战。替代梦塔防"后期兵超级强"的强制结束机制，但更温和。

### 8.3 单局情绪曲线设计目标

```
0:00 ─ 建造期    双方放置建筑，出小兵，试探
1:30 ─ 第一波    双方兵力在地图中段相遇，第一次交战
3:00 ─ 资源加速  节奏加快，开始用法术，兵力密度上升
5:00 ─ 危机期    某一方基地开始受到威胁
7:00 ─ 决战期    资源丰沛，大量单位，正面决战
≤10:00 ─ 结束   基地血量归零
```

## 九、MVP 范围边界

### 包含（必须做）

- [x] 基地自动产金币 + 金币上限
- [x] 6 张手牌系统，12 张牌池
- [x] 3 种兵种（普通兵、盾兵、弓箭兵）+ 行为差异
- [x] 2 种建筑（兵营、箭塔）
- [x] 2 种法术（急速冲锋、陨石打击）
- [x] 地图：2 条路线 + 4 个建筑格
- [x] 主基地 HP，归零判负
- [x] 时间加速机制（防拉锯）
- [x] AI 对手（模拟真人行为）

- [ ] 真实联机对战（帧同步）
- [ ] 建筑升级
- [ ] 更多兵种、建筑、法术
- [ ] 段位/排行榜
- [ ] 卡牌收集/养成系统
- [ ] 多张地图
- [ ] 音效和粒子特效（先用占位）

## 十、待确认的设计问题（需要在原型阶段验证）

1. **金币上限 30** 是否合适？太低会让玩家感觉"钱不够用"，太高会弱化上限的逼迫感。
2. **盾兵的仇恨吸引**在小尺寸战场上是否能被玩家感知到？
3. **弓箭兵停下来射击**的行为，是否会在实际战斗中造成它们被夹击的情况，导致弓箭兵体验很挫？
4. **两条路线**的分兵压迫是否在 5-10 分钟单局内有足够时间展开？还是应该 MVP 先只做一条路线？
5. **AI 难度**：AI 应该模仿"中等玩家"还是"有明显失误的新手玩家"？
