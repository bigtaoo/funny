## 1. 货币模型回顾

- **单一货币 `coins`**，只能**看广告 / 充值**获得（不靠打关卡产出）。服务器权威，绝不可刷（`META_DESIGN.md §2`）。
- **消耗**：商店直购物品/皮肤、开盲盒。
- **PvE 养成不花 coins**，花**关卡掉落材料**（§5）。
- 物品三来源：关卡奖励 / 商店直购 / 盲盒（稀有唯一来源）。

## 4. 盲盒概率与保底 — DRAFT

### 4.1 稀有度权重（标准池）
| 稀有度 | 权重 | 概率 |
|---|---|---|
| common | 700 | 70% |
| rare | 230 | 23% |
| epic | 60 | 6% |
| legendary | 10 | **1%** |

### 4.2 保底（pity）
- **十连保底**：每 10 抽至少 1 个 epic 及以上。
- **大保底**：累计 **90 抽**必出 legendary（`gacha.pity[poolId]` 计数，出后清零）。
- 保底计数服务端维护，逐抽落 `gachaHistory`（`SERVER_API.md §2.4`）。

### 4.3 重复转化（`dupePolicy`）
| 抽到已拥有 | 转化 |
|---|---|
| common/rare | 退 **碎片**（攒满兑换指定皮肤） |
| epic/legendary | 退 **coins**（按稀有度，如 legendary 退 1,500） |

> 碎片兑换表与退币额待定（§7）。

## 6. F2P 进度推演 — DRAFT

以广告每日上限算无氪天花板：

```
每日 coins 上限 = 5 次 × 50 = 250 coins/天
单抽 150 coins → 约 1.6 抽/天；十连 1,350 → 约 5.4 天攒一次
大保底 90 抽 = 13,500 coins → 无氪约 54 天保底出一个 legendary
一个 rare 皮肤 800 → 约 3.2 天直购
```

| 玩家 | 节奏感受 |
|---|---|
| 无氪 | 每周末攒够一次十连；legendary 靠长期保底 / 活动 → 有奔头不绝望 |
| 微氪（首充+小档） | 立刻可十连 1~2 次，体验"开盒爽点" |
| 中氪（98档） | 单次充值 ≈ 一个 legendary 保底区间，目标清晰 |

> 调参目标：无氪有稳定可期的获取曲线（不劝退），付费提供"加速 + 爽点"，**绝不影响 PvP 公平**。
