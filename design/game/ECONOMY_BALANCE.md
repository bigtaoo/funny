## 1. 货币模型回顾

**两套互不相通的经济，名字不再撞：**

| | 局内资源 `ink`（墨滴） | 元货币 `coins`（金币） |
|---|---|---|
| 形态 | 单局内回收，打完清零；图标=墨滴，寓意「用墨绘制角色/建筑」 | 跨局持久，服务器权威，绝不可刷（`META_DESIGN.md §2`） |
| 产出 | 按时间自动回（`config.ts COIN_REGEN_BASE`，加速档抬高），升基地 +回速 | 看激励广告 / IAP 充值 / **称号系统**（天梯段位）/ **成就系统**（统计里程碑） |
| 消耗 | 出牌（兵/建筑/法术）、升基地 | 盲盒 / 直购关卡材料 / 部分 common 皮肤 |
| 确定性 | 是（不碰存档，进 PvP 锁步） | 否（服务端事务） |

> ⚠️ 代码里局内资源**待重命名 `coins → ink`**（`config.ts` 常量 `COIN_*`、i18n、HUD、StickmanRuntime 贴图无关）。两词同名是历史遗留，新文案/图标一律按上表区分。

**金币（coins）的产出与消耗 — 主线**

```
产出: 广告(cap) + IAP + PvP天梯成就 ──┐
                                      ├─► 金币 ──┬─► 盲盒（稀有唯一来源）
消耗主 sink = 关卡养成 ───────────────┘           ├─► 直购关卡材料 → PvE 升级（受硬墙保护，永不进 PvP）
                                                  └─► 直购部分 common 皮肤
```

- **金币能买材料 → 喂 PvE 升级**：花钱只加速单机养成，**绝不影响 PvP 公平**（硬墙在 `META_DESIGN.md §5.2`）。材料仍可纯靠打关卡免费获取（§5），直购只是加速。
- **PvP 不直接产物品**，而是天梯积分到阶 → 成就解锁 → 奖励金币（类似成就系统），给 PvP 一条游戏内的金币 faucet。
- **金币游戏内 faucet = 称号 + 成就两套**（与广告/IAP 并行，详见 §2.3 / §2.4）：
  - **称号系统**：由 PvP 天梯**段位**驱动，首次达到某段位发一次性金币（不可重复刷 → 天然控量）。
  - **成就系统**：统计里程碑（如「累计击杀 N 个弓箭手」「通关第 X 章」），分阶解锁、每阶一次性发金币。
- 物品来源：关卡奖励（材料）/ 商店直购（金币）/ 盲盒（稀有唯一来源）。
- **平衡铁律**：金币整体产出/消耗须严控（§6 推演），主 sink 压在关卡养成而非纯抽卡，避免通胀。称号/成就均为**一次性**发放，避免成为无限金币泵。

### 2.3 称号系统（天梯段位 → 一次性金币）— DRAFT

PvP 天梯按段位授予称号；**首次**达到某段位发一次性金币（再次升降不重复发 → 天然控量）。

| 段位（称号） | 首达金币 | 累计 |
|---|---|---|
| 见习（起始） | 0 | 0 |
| 青铜 | 100 | 100 |
| 白银 | 200 | 300 |
| 黄金 | 400 | 700 |
| 铂金 | 700 | 1,400 |
| 钻石 | 1,200 | 2,600 |
| 大师 | 2,000 | 4,600 |
| 宗师 | 3,000 | 7,600 |

> 爬满全段位一次性 ~7,600 金币 ≈ 半个 legendary 保底；一次性、不可刷。**赛季重置奖励**（按赛季峰值段位重复发，额度更小）留待后期，初期只做首达。

### 2.4 成就系统（统计里程碑 → 一次性金币）— DRAFT

与天梯无关的统计成就，分 I/II/III 阶解锁，每阶一次性发金币。下为**模板初值**（具体条目后期扩充）：

| 成就示例 | 阶 I / II / III 阈值 | 阶 I / II / III 金币 |
|---|---|---|
| 累计击杀弓箭手 | 100 / 500 / 2,000 | 50 / 100 / 200 |
| 累计击杀守卫 | 100 / 500 / 2,000 | 50 / 100 / 200 |
| 释放陨石次数 | 20 / 100 / 400 | 50 / 100 / 200 |
| 通关章节数 | 1 / 3 / 全部 | 100 / 200 / 400 |
| 累计 PvP 胜场 | 10 / 50 / 200 | 50 / 150 / 300 |

> 单条成就满阶 ~350 金币；按 ~25 条规划 → 全游戏一次性 ~8–9k 金币池。**全部一次性**，不构成持续金币泵。条目与阈值后期慢慢加。

## 5. 材料 + PvE 养成 — DRAFT

材料来源：**关卡掉落**（免费）+ **金币直购**（加速，见 §1）。材料 sink **方向：装备系统 / 锻造**——用材料锻造装备，装备给单位提供加成（替代/承载原「直接 stat 升级树」§5.3 的效果，具体演进后期定）。

> ⚠️ **硬墙关系（重要）**：装备**现阶段只用于 PvE**，受 `META_DESIGN.md §5.2` 硬墙保护、永不进 PvP → 金币买材料锻装备仍是「纯单机加速」，不破 PvP 公平。**但后期装备会用于 SLG 的 PvP 战力**（§8）——届时装备即 PvP 战力，养成不再算纯单机加速，需重新设计该玩法的公平/匹配/付费边界。

## 6. F2P 进度推演 — DRAFT

以广告每日上限算无氪天花板：

```
每日 coins 上限 = 5 次 × 50 = 250 coins/天
单抽 150 coins → 约 1.6 抽/天；十连 1,350 → 约 5.4 天攒一次
大保底 90 抽 = 13,500 coins → 无氪约 54 天保底出一个 legendary
一个 rare 皮肤 800 → 约 3.2 天直购
```

| 玩家 | 节奏感受 |
|---|---|
| 无氪 | 每周末攒够一次十连；legendary 靠长期保底 / 活动 → 有奔头不绝望 |
| 微氪（首充+小档） | 立刻可十连 1~2 次，体验"开盒爽点" |
| 中氪（98档） | 单次充值 ≈ 一个 legendary 保底区间，目标清晰 |

> 调参目标：无氪有稳定可期的获取曲线（不劝退），付费提供"加速 + 爽点"，**绝不影响 PvP 公平**。

### 6.1 鲸鱼天花板（设计目标）

- **约 €2,000 / ¥20,000 拿到 ~90% 游戏物品** 视为可接受上限——即全图鉴的「天花板」定在中高位，不做无底洞式抽卡。
- 含义：盲盒池总量 + 直购皮肤定价反推后，重氪一次性清掉绝大多数内容是 OK 的（留 ~10% 给限时/赛季内容做长尾）。
- 这条是定价/池子总量的**锚**：legendary 数量 × 保底成本 + epic/rare 皮肤直购总价应大致收敛到此区间。
