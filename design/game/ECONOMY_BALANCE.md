## 1. 货币模型回顾

**两套互不相通的经济，名字不再撞：**

| | 局内资源 `ink`（墨滴） | 元货币 `coins`（金币） |
|---|---|---|
| 形态 | 单局内回收，打完清零；图标=墨滴，寓意「用墨绘制角色/建筑」 | 跨局持久，服务器权威，绝不可刷（`META_DESIGN.md §2`） |
| 产出 | 按时间自动回（`config.ts COIN_REGEN_BASE`，加速档抬高），升基地 +回速 | 看激励广告 / IAP 充值 / **称号系统**（天梯段位）/ **成就系统**（统计里程碑） |
| 消耗 | 出牌（兵/建筑/法术）、升基地 | 盲盒 / 直购关卡材料 / 部分 common 皮肤 |
| 确定性 | 是（不碰存档，进 PvP 锁步） | 否（服务端事务） |

> ✅ 代码里局内资源 `coins → ink` 重命名**已完成**（2026-06-13，纯重构不改数值）。新文案/图标一律按上表区分。

**金币（coins）的产出与消耗 — 主线**

```
产出: 广告(cap) + IAP + PvP天梯成就 ──┐
                                      ├─► 金币 ──┬─► 盲盒（稀有唯一来源）
消耗主 sink = 关卡养成 ───────────────┘           ├─► 直购关卡材料 → PvE 升级（受硬墙保护，永不进 PvP）
                                                  └─► 直购部分 common 皮肤
```

- ⚠️ **文档-代码漂移修正（2026-07-03 核查）**：本节原写「金币直购材料 → 喂 PvE 升级」**在代码里不存在**——`shared/economy.ts SHOP_ITEMS` 只售皮肤 + 强化保护道具，无任何材料/文具直购项；材料来源仅关卡掉落 / 高稀抽卡 / 装备分解。**金币→战力的真实路径是绕道盲盒**（金币→盲盒→抽到卡→喂卡升级），外加装备强化直扣金币（`equipment.ts enhanceCost = 40×(lv+1)`，失败也扣）。**角色卡升级不消耗金币**（只吃别的卡的 XP，70% 转移）。若要恢复「金币直购材料」需新开 sink 并入 §6 预算，当前未实装。硬墙（装备战力只进 PvE，`META_DESIGN.md §5.2`）不受影响。
- **PvP 不直接产物品**，而是天梯积分到阶 → 成就解锁 → 奖励金币（类似成就系统），给 PvP 一条游戏内的金币 faucet。
- **金币游戏内 faucet = 称号 + 成就两套**（与广告/IAP 并行，详见 §2.3 / §2.4）：
  - **称号系统**：由 PvP 天梯**段位**驱动，首次达到某段位发一次性金币（不可重复刷 → 天然控量）。
  - **成就系统**：统计里程碑（如「累计击杀 N 个弓箭手」「通关第 X 章」），分阶解锁、每阶一次性发金币。
- 物品来源：关卡奖励（材料）/ 商店直购（金币）/ 盲盒（稀有唯一来源）。
- **平衡铁律**：金币整体产出/消耗须严控（§6 推演），主 sink 压在关卡养成而非纯抽卡，避免通胀。称号/成就均为**一次性**发放，避免成为无限金币泵。

#### 5.5.1 战力来源权重 + 装备占比上限（2026-06-14 DRAFT 初定）

装备改 PvE 战力 → 必须和关卡难度曲线挂钩，否则要么卡死玩家、要么装备过强使关卡失去意义。先定旋钮：

| 战力来源 | 占 PvE 有效战力（DRAFT） | 说明 |
|---|---|---|
| 基地升级（已有）+ 卡组 + 操作 | **~65%** | 无装备玩家的"地板"，能稳过早/中期关卡 |
| 装备（合成养成） | **上限 ~35%** | 后期关卡的"门槛"；满装与无装的战力差封顶约 1.5× |

- **设计目标**：早/中期关卡用地板战力即可通；**后期关卡按"需大部分可得装备战力才能稳过"调**，使"无装备通不了后期"成立——但封顶 35% 保证无装备玩家不归零（不是 0 输出的死局，而是"打得动但过不去硬关"）。
- **装备作用对象**：初期挂在玩家的**全局小队 loadout**（影响全军，轻量、美术少），`equipped` 存档结构预留**按兵种独立装备**的扩展位（后期养成深度 + SLG 用）。
- **材料（文具）产出节奏**：沿用 §5.2 关卡掉落 + §5.4 每日副本；合成配方按 §5.5 的 9 级结构（Lv1 一件 ≈ 5 个同种基础文具，对齐"一件衣服=5铅笔+1橡皮"的体感）。具体掉率/配方待模拟调参（§7）。

## 6. F2P 进度推演 — DRAFT

无氪日金币天花板（广告现为小头，大头在日常/任务/活动）：

```
广告 = 5 次 × 10 = 50 coins/天（小头）
+ 日常/任务/活动 ~165 coins/天（§4.2 估算）
≈ 无氪日上限 ~215 coins/天
单抽 150 coins → 约 1.4 抽/天；十连 1,350 → 约 6.3 天攒一次
大保底 90 抽 = 13,500 coins → 无氪约 63 天保底出一个 legendary

```

| 玩家 | 节奏感受 |
|---|---|
| 无氪 | 每周末攒够一次十连；legendary 靠长期保底 / 活动 → 有奔头不绝望 |
| 微氪（首充+小档） | 立刻可十连 1~2 次，体验"开盒爽点" |
| 中氪（98档） | 单次充值 ≈ 一个 legendary 保底区间，目标清晰 |

> 调参目标：无氪有稳定可期的获取曲线（不劝退），付费提供"加速 + 爽点"，**绝不影响 PvP 公平**。

## 8. SLG（暂缓 / 未来玩法 · 2026-06-14 收敛定义）

> 决策 **(b)**：SLG **暂不做**，但**必须预留结构空间**——装备数值与 `equipped` 存档结构现在就按"将来能进 SLG"设计，不写死纯 PvE。

- **SLG = 玩家之间的数值消耗战场**：夺取领地、相互杀伐、封官加爵、称霸一方。它**天生就是 PvP**，所以旧称"SLG PvP"作废，统一叫 **SLG**。
- **定位 = 赚钱区，不要求公平**：养成战力 = 付费战力，这里就是让玩家用数值（含 PvE 锻造的装备）相互碾压/消耗的地方。
- **与天梯/知己 PvP 严格分开**：天梯/知己永远公平（装备归一、拼操作），SLG 装备生效（拼养成）——**两者分开匹配、分开榜**，互不污染。
- 依赖足够玩家基数 + 稳定营收才启动；前期不实现，仅登记方向 + 预留数据结构（不预埋复杂玩法逻辑）。
