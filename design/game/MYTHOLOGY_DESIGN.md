# Notebook Wars — 东西方神话世界观（储备方案）

> **状态：设计储备，暂不实装。**  
> 现行战役走**武侠世界观**（宗门 / 家族 / 陶 & Anna 主线，见 `CAMPAIGN_STORY.md`）。  
> 本文档记录「东西方神话碰撞」世界观方案，供后期专项切换或扩展章节（如 DLC / 神话赛季）使用。  
> 创建：2026-06-17。

---

## 0. 核心隐喻

一个叛逆少年的笔记本——他同时迷恋东方神话（《山海经》/ 封神 / 道经）和西方神话（希腊 / 北欧 / 凯尔特），懒得在乎文化纯正，把两套体系混进同一个战场。

```
东方  ──────装订线────── 西方
天庭 / 山海经           奥林匹斯 / 北欧神殿
神兵天将                希腊英雄 / 北欧战士
精怪异兽                神话怪物
```

**装订线就是战场。**  
东方是「铅笔草稿」风格（线条粗粝、墨迹飞溅），西方是「钢笔细描」风格（几何严整、描边锐利），两种笔触在同一页上共存——与现有手绘美术语言天然衔接。

---

## 1. 为什么契合「叛逆少年」

| 现行武侠世界观 | 神话碰撞世界观 |
|---|---|
| 宗门 / 家族 / 比武——有序的秩序感 | 神与怪物——打破一切秩序的存在 |
| 主角是少年学徒，在体制内成长 | 主角召唤的东西本身就是「离经叛道者」（朱雀出逃天庭、哈耳庇厄逃离宙斯） |
| 东方视角单一 | 东西混搭：不屑于「只信一套神话」，这正是叛逆的表达方式 |

**哪吒 + 阿喀琉斯在同一支队**——这种荒谬感恰恰是少年想象力最真实的样子。

---

## 2. 派系结构

### 2.1 东方：天庭军 × 山海经诸怪

| 分类 | 代表形象 | 战场角色 |
|---|---|---|
| 天兵将 | 哪吒、李靖（托塔天王）、杨戬 | 正规军，纪律严明 |
| 山海经异兽 | 穷奇、饕餮、獬豸、鲲鹏 | 混沌力量，体型极端 |
| 仙道散修 | 白蛇、九尾狐、金乌 | 游击奇兵，特效多 |

### 2.2 西方：奥林匹斯 × 北欧九界

| 分类 | 代表形象 | 战场角色 |
|---|---|---|
| 希腊英雄 | 阿喀琉斯（步兵原型）、奥德修斯（弓手原型）、忒修斯 | 人类精英，数值稳定 |
| 希腊怪物 | 独眼巨人、美杜莎、弥诺陶洛斯、哈耳庇厄 | 怪种，拥有特殊 trait |
| 北欧 | 瓦尔基里（飞行）、霜巨人（重甲）、德鲁格尔（不死） | 极端特性单位 |

## 4. 飞行单位：纸飞机的神话身份

**外观**：折纸形态——程序草稿骨骼，三角翼 + 轻微上下浮动动画，符合现有笔记本视觉语言。

### 4.1 东方版：朱雀

- **原型**：南方神兽，凤凰族裔，火焰之鸟
- **叙事**：从天庭出逃的离经叛道者，不服天规，自愿投靠少年军
- **视觉**：铅笔勾翅，尾部 2–3 条扭曲火焰线（SketchPen stroke，不用粒子）
- **PvP 卡名**：朱雀（Vermilion Bird）

### 4.2 西方版：哈耳庇厄 Harpy

- **原型**：希腊神话鹰翼女妖，迅捷凶猛，宙斯的差使
- **叙事**：从奥林匹斯逃离宙斯管辖，与朱雀是宿命对手
- **视觉**：钢笔细描翅膀，人形轮廓，线条锐利
- **PvP 卡名**：哈耳庇厄（Harpy）

> **实装建议**：先出朱雀作为 PvP 飞行单位；哈耳庇厄作为后续 DLC / 神话赛季的第二张飞行牌。  
> 两张牌机制完全相同（见下方数值参考），皮肤不同，对应东西两派收集系统。

### 4.3 飞行单位数值参考（草案）

| 参数 | 值 | 备注 |
|---|---|---|
| HP | 45 | 脆——1–2 个弓箭手可清 |
| 速度 | 1.4 | 快于步兵，慢于 Runner |
| 攻击 | 12 | 中等，不算主力输出 |
| 攻击间隔 | 1.0s | 标准 |
| 射程 | 1 | 近战（只能打相邻） |
| 费用 | 11 墨 | 高于步兵，体现反制稀缺性 |
| `flying` | true | 地面单位无法寻敌 |
| `canTargetFlying` | false | 自身不打飞行目标 |

## 5. 章节神话主题框架（六章版）

> 与现行武侠章节结构并列储备，可用于替换或扩展。

| 章节 | 神话体系 | 地图风格 | 代表 Boss |
|---|---|---|---|
| Ch1 | 山海经入门 | 荒野 / 异兽图志页 | 穷奇（重甲 Boss） |
| Ch2 | 天庭冲突 | 云端 / 天宫走廊 | 哪吒（高速 Boss，死亡分裂为莲花） |
| Ch3 | 希腊入侵 | 石柱废墟 / 爱琴海边 | 弥诺陶洛斯（Ironclad 变体） |
| Ch4 | 北欧南下 | 雪原 / 世界树根部 | 火巨人 Surtr（岩浆 hazard 关） |
| Ch5 | 两方联手围剿 | 装订线战场（混合笔触） | 奥丁使者 + 天庭上将联合军 |
| Ch6 | 终章 · 装订线决战 | 少年的书桌（最后一页） | 融合神祇（东方甲胄 + 西方神器） |

**Ch6 终 Boss 设计草案**：  
「执雷电权杖的四臂战神」——外形东方重铠（铅笔线条）+ 西方神器（钢笔描边），象征少年内心两种文化的最终融合而非对立。机制上同时拥有死亡分裂（莲花 + 雷电球）和飞行阶段切换。

## 6. 世界观与美术方向的衔接

| 美术规则（`art-direction.md`） | 神话世界观对应 |
|---|---|
| 铅笔 / 钢笔 / 马克笔三笔触 | 东方怪兽用铅笔草稿，西方英雄用钢笔细描，两者 boss 用马克笔重涂 |
| 阵营蓝红 | 玩家（少年一侧）= 蓝，敌方入侵者 = 红，不分东西 |
| 手绘笔记本纸背景 | 每章扉页是一页涂满笔记的纸，章节 boss 是从页边飞出来的 |
| SketchPen 程序绘制特效 | 死亡特效 = 笔迹消散；飞行单位出场 = 折纸展翅动画 |
