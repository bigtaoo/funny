# Notebook Wars — 东西方神话层（神力赋予方案）

> **状态：方向已定稿（2026-06-19），代码未动。**  
> 神话**不替换**武侠世界观，也**不是**独立的神魔阵营战争。它是叠加在现有战役阵容（陶 & Anna 的六个角色）之上的**「神力赋予」层**——原型 + 神力 + 画稿。武侠是骨（角色、信任主线、宗门家族一字不改），神话是皮（作者赋予角色的神力 / 美术 / 技能）。  
> 关联铁律见 `world.md`（陶 / Anna 文化背景与两个本子）/ `CAMPAIGN_STORY.md`。  
> 创建：2026-06-17，定稿：2026-06-19。

## 0. 核心隐喻：两个本子相遇

这个故事里有**两个笔记本**，各自纯粹：

```
陶的本子  ──────装订线────── Anna 的本子
中国孩子，来德两年            德国世家子女
只画东方神话                  只画西方神话
（山海经 / 封神 / 道经）       （希腊 / 北欧 / 凯尔特）
铅笔草稿（线条粗粝、墨迹飞溅）   钢笔细描（几何严整、描边锐利）
```

**东西碰撞不是某个孩子兼收并蓄，而是两个本子相遇才涌现出来的。**装订线 = 两本摞在一起的那条缝（呼应 `CAMPAIGN_STORY.md` 尾声「父亲把两个本子摞在一起读」）。陶只画东方、Anna 只画西方，碰在一起才有了东西同场——结构产生混搭，不是性格产生混搭。两种笔触与现有手绘美术语言天然衔接。

## 1. 神话怎样进入游戏：作者赋予神力

神话**不**让战役后期变成神魔大战。它进入游戏的方式是——

> **作者给自己本子里的角色，赋予某个神话角色的神力。**

李川还是李川，但陶给了他「哪吒之力」（风火轮、火尖枪）。神话角色 = 赋予到既有角色身上的**原型 + 神力 + 画稿**，不是替换那个角色。

| 这把三件事焊死 | 说明 |
|---|---|
| **武侠是骨** | 六个孩子、信任主线、宗门家族——一字不改 |
| **神话是神力层** | 原型/神力/皮肤/技能，由作者赋予到既有角色 |
| **成长有戏内解释** | 两个孩子画得越久、想象力越大，才开始给笔下的人赋予神力——这正是神话层「长大后才出现」的纵向叠加 |
| **PvP 全角色通用** | 玩家就像作者，自由选择把哪种神力赋予谁，不受叙事阵营限制 |

陶与 Anna 本人**不进神话、不当神**，他们留在真实层当作者，神力是他们赋予角色的，作者意象本身不带入故事。

## 3. 六角镜像原型表（神力锚点 + 美术方向）

神话角色既是六个主角的**性格内核罗盘**，也是画师的**参考像**。战役按镜像结构写（方家 ↔ Hartmann，同位反性格），神力层继承这个镜像：**每对同位角色 = 同一原型，东西两种表达，相反的温度**。

阵营按出身（= 本子归属）：方家 = 东方（陶的本子·铅笔草稿），Hartmann = 西方（Anna 的本子·钢笔细描）。

| 镜像对 | 方家·东方 | Hartmann·西方 | 共享原型 / 内核 |
|---|---|---|---|
| 先锋 | **李川 → 哪吒**：风火轮野火，待驯的冲（弧光=学会克制） | **Max → 阿喀琉斯**：磨好的冷锋，一开始就在克制里；阿喀琉斯之踵=「确信即软肋」的角色钩子 | 无双前锋——一个是野火，一个是磨好的刀刃 |
| 守护 | **陈守 → 玄武**：龟蛇玄甲，镇守北方；名字就是「守」，本能地走在前面替人挡 | **Lena → 雅典娜**：神盾 Aegis，算过才站在盾位的智性防守（「不是牺牲，是选择」） | 挡在前面的人——本能 vs 计算 |
| 洞察 | **苏远 → 后羿**：射日之眼，看得准，箭不早不晚 | **Mara → 奥德修斯**：带着一个问题的漫长归途，终得其解（答案=信任） | 读懂下一步的人——直觉的准 vs 求解的智 |

> **取舍记录**：Max 在阿喀琉斯 / 提尔 / 斯巴达之间选阿喀琉斯——同为「无双前锋」原型，与李川=哪吒形成「待驯的火 vs 磨好的锋」温度反差；且阿喀琉斯之踵给「骄傲即确信」的 Max 一个天然性格钩子（唯一盲点恰恰来自确信）。  
> 朱雀已占用为 PvP 飞行单位（见 §4），四神只取玄武给陈守，不强凑全套。

**非具名单位（PvE 杂兵 / 怪种）的神话化身**（供地图与敌方阵容取用）：

| 现有 UnitType | 东方化身 | 西方化身 |
|---|---|---|
| Infantry | 天兵（托塔天王麾下卒） | 希腊重步兵 Hoplite |
| Archer | 后羿门徒 | 阿波罗弓手 |
| ShieldBearer | 夜叉金刚 | 斯巴达盾卫 |
| Ironclad（PvE） | 穷奇（凶兽，抵御箭矢） | 独眼巨人 Cyclops |
| Runner（PvE） | 獬豸（疾蹄异兽） | 地狱犬 Cerberus |

## 4. 飞行单位：纸飞机的神话身份

**外观**：折纸形态——程序草稿骨骼，三角翼 + 轻微上下浮动动画，符合现有笔记本视觉语言。

### 4.1 东方版：朱雀

- **原型**：南方神兽，凤凰族裔，火焰之鸟
- **叙事**：从天庭出逃的离经叛道者，不服天规，自愿投靠少年军
- **视觉**：铅笔勾翅，尾部 2–3 条扭曲火焰线（SketchPen stroke，不用粒子）
- **PvP 卡名**：朱雀（Vermilion Bird）

### 4.2 西方版：哈耳庇厄 Harpy

- **原型**：希腊神话鹰翼女妖，迅捷凶猛，宙斯的差使
- **叙事**：从奥林匹斯逃离宙斯管辖，与朱雀是宿命对手
- **视觉**：钢笔细描翅膀，人形轮廓，线条锐利
- **PvP 卡名**：哈耳庇厄（Harpy）

> **实装建议**：先出朱雀作为 PvP 飞行单位；哈耳庇厄作为后续 DLC / 神话赛季的第二张飞行牌。  
> 两张牌机制完全相同（见下方数值参考），皮肤不同，对应东西两派收集系统。

> **澄清（2026-07-07）**：本条的「哈耳庇厄 Harpy」是 **PvP 折纸纸飞机飞行卡**（HP 45 / 速 1.4，见 §4.3）。它与 Anna 侧的 PvE 复用兵 **Aello（鹰翼女妖，HP 22 / 速 2.2）** 是**两个不同实体**，形象/数值均不同（见 [`ANNA_CHARACTERS.md`](ANNA_CHARACTERS.md#aello-鬼鸟harpy)），两者仅共用「哈耳庇厄／Harpy」这一神话名，无需对齐数值。

### 4.3 飞行单位数值参考（草案）

| 参数 | 值 | 备注 |
|---|---|---|
| HP | 45 | 脆——1–2 个弓箭手可清 |
| 速度 | 1.4 | 快于步兵，慢于 Runner |
| 攻击 | 12 | 中等，不算主力输出 |
| 攻击间隔 | 1.0s | 标准 |
| 射程 | 1 | 近战（只能打相邻） |
| 费用 | 11 墨 | 高于步兵，体现反制稀缺性 |
| `flying` | true | 地面单位无法寻敌 |
| `canTargetFlying` | false | 自身不打飞行目标 |

## 5. 章节神话主题框架（六章版）（深层储备，非当前方向）

> **注（2026-06-19）**：本节是「神话替换武侠章节」的早期储备，**已被 §1「神力赋予」层取代**——当前战役章节结构沿用 `CAMPAIGN_STORY.md` 的武侠六章，神话不另起一套神魔章节。下表的地图风格与 Boss 形象可作为 PvE / 赛季内容的素材库。

| 章节 | 神话体系 | 地图风格 | 代表 Boss |
|---|---|---|---|
| Ch1 | 山海经入门 | 荒野 / 异兽图志页 | 穷奇（重甲 Boss） |
| Ch2 | 天庭冲突 | 云端 / 天宫走廊 | 哪吒（高速 Boss，死亡分裂为莲花） |
| Ch3 | 希腊入侵 | 石柱废墟 / 爱琴海边 | 弥诺陶洛斯（Ironclad 变体） |
| Ch4 | 北欧南下 | 雪原 / 世界树根部 | 火巨人 Surtr（岩浆 hazard 关） |
| Ch5 | 两方联手围剿 | 装订线战场（混合笔触） | 奥丁使者 + 天庭上将联合军 |
| Ch6 | 终章 · 装订线决战 | 少年的书桌（最后一页） | 融合神祇（东方甲胄 + 西方神器） |

**Ch6 终 Boss 设计草案**：  
「执雷电权杖的四臂战神」——外形东方重铠（铅笔线条）+ 西方神器（钢笔描边），象征少年内心两种文化的最终融合而非对立。机制上同时拥有死亡分裂（莲花 + 雷电球）和飞行阶段切换。
